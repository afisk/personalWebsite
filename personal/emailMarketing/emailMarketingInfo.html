Project Inspiration

A lot of sales work is done by reaching out to multiple potential clients all at once via generated contact lists that can be purchased from online sources. I was contacted by a former work associate, Scott Krause, and asked to build an app that could send marketing emails from a GMail account. There are many other applications already available on the market, however they are a costly investment that must be paid every single time an email is sent. This application logs Scott into his GMail through a Google Auth application. This allows him to generate emails and send them all from his personal/work email as many times as he would like with no additional charges.

This application was built on python with PySimpleGUI for the user interface construction. This leverages the simplegmail application for the Google O-Auth login. This then use a list of contacts in a .csv file to populate the values in the marketing emails (see below for exapmles). It can be used with any .csv. The user just needs to type in the header value, but surrounded by double curly braces and the marketing application will substitute in the value with a contact's personal information from the .csv file.
